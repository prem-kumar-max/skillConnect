# 🚀 SkillConnect

SkillConnect is a full-stack web platform that connects people based on their skills.  
Users can learn, collaborate, showcase their expertise, and offer services — all in one place.

---

## 🌟 Problem Statement

Currently, users rely on multiple platforms for:
- Learning (YouTube, courses)
- Networking (LinkedIn)
- Freelancing (Upwork, Fiverr)

👉 SkillConnect solves this by combining all these features into a single platform.

---

## 💡 Features

### 👤 User Features
- User Registration & Login
- Profile Creation (Skills, Bio, Portfolio)
- Search for skilled professionals
- View detailed user profiles
- Create different types of posts:
  - Articles
  - Projects
  - Reviews
- Connect / Hire users

---

### 🧑‍💻 Dashboard Features
- Personal dashboard for each user
- Create, edit, and delete posts
- Manage profile and skills
- View activity and connections

---

### 🔍 Search System
- Search users based on skills
- Results displayed as profile cards
- Public access for browsing
- Login required for interaction

---

### 💳 Payment Integration
- Premium subscription plans
- Integration with:
  - Stripe (International users)
  - Razorpay (Indian users)
- Secure checkout and verification

---

### 👨‍💼 Admin Features
- Admin dashboard
- Manage users
- Monitor posts
- Handle reports and moderation
